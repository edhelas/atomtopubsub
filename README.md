## About AtomToPubsub

AtomToPubsub is a simple Python script that parses Atom feeds and pushes
the entries to a designated [XMPP Pubsub Node](http://xmpp.org/extensions/xep-0060.html)

## Requirements

* Python >= 3.5
* GIT Core
* PIP
* feedparser
* jsonpickle
* slixmpp
* screen
